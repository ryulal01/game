# Endless Runner Game

## Игрок

* **Автоматическое движение вперёд** — игрок постоянно движется без нажатия клавиш.  
* **Перемещение влево и вправо** — можно менять позицию или линии движения.  
* **Прыжки и двойной прыжок** — один прыжок в воздухе, чтобы перепрыгивать препятствия.  
* **Здоровье:**  
  * Отображается максимальное и текущее здоровье.  
  * Получает урон при столкновении с препятствиями.  
  * Игра начинаеся заново, если здоровье достигает нуля.
  

---

## Препятствия

* **Бесконечная генерация** — появляются на пути игрока по мере движения.  
* **Два типа блоков: они наносят разных по величине урон**    
* **Удаление пройденных препятствий** — объекты автоматически удаляются после прохождения, чтобы оптимизировать работу игры.
* **Препятсвия будут рапологаться случайно
* **Тип препятсвия выбирается случайно

---

## Алгоритм работы игры

* **Инициализация сцены:**  
  * Загрузка игрока, препятствий и интерфейса.  
  * Настройка генератора препятствий.  
* **Игровой цикл:**  
  * Автоматическое движение игрока вперёд.  
  * Обработка ввода (влево/вправо, прыжок, двойной прыжок).  
  * Проверка столкновений с препятствиями:  
    * В случае столкновения игрок теряет здоровье.  
    * Если здоровье ≤ 0, игра начинается заново.  
  * Генерация новых препятствий по мере продвижения.  
  * Удаление пройденных препятствий для оптимизации.  
* **Обновление интерфейса:**  
  * Отображение текущего здоровья.  

---

## Реализация

* **Основная сцена:** `MainScene.unity`  
* **Главные скрипты:**  
  * `PlayerController.cs` — управление игроком, прыжки.
  * `PlayerHealth.cs` - здоровье  
  * `ObstacleManager.cs` — генерация и удаление препятствий.
  * `Obstacle.cs` - управляет поведением препятствия в игре
  * `GameManager.cs` — контролирует общую логику игры
## Точки входа
* **MainScene.unity:**	Загружает все объекты и запускает игру.
* **GameManager:**	Инициализация логики игры, рост сложности.
